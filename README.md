<h2 style="color:#111827; font-family:Segoe UI, system-ui;">🚀 Instrucciones de ejecución</h2>
<ol style="color:#374151; font-size:14px; margin-left:20px;">
  <li>
    <strong>Instalar dependencias</strong><br />
    Verificar el archivo <code>requirements.txt</code> e instalar las librerías:
    <ul style="margin-top:4px; margin-left:20px;">
      <li>Pandas</li>
      <li>Streamlit</li>
      <li>Numpy</li>
      <li>Matplotlib</li>
      <li>Seaborn</li>
    </ul>
    <pre style="background:#0b1120; color:#e5e7eb; padding:10px; border-radius:6px; font-size:13px;">
pip install -r requirements.txt</pre>
  </li>
  <li style="margin-top:8px;">
    <strong>Ejecutar la aplicación</strong><br />
    En la consola (CMD), ejecutar:
    <pre style="background:#0b1120; color:#e5e7eb; padding:10px; border-radius:6px; font-size:13px;">
streamlit run app.py</pre>
  </li>
  </li>
  <li style="margin-top:8px;">
    <strong>NOTA: El archivo imagen_home.jpg debe encontrarse en la misma carpeta que la aplicación (app.py)</strong><br />
  </li>
</ol>
