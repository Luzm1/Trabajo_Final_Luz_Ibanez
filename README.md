<h2 style="color:#111827; font-family:Segoe UI, system-ui;">📌 Descripción</h2>
<p style="color:#374151; font-size:14px;">
  Este proyecto es el trabajo final del curso <strong>Especialización Python for Analytics</strong>, el cual está orientado al Análisis Exploratorio de Datos (EDA) del dataset  TelcoCustomerChurn.csv, aplicando todos los conocimientos adquiridos en el lenguaje con el fin de dar apoyo en la toma de decisiones.
</p>

<h2 style="color:#111827; font-family:Segoe UI, system-ui;">📁 Estructura del proyecto</h2>
<ul style="color:#374151; font-size:14px; list-style-type:square; margin-left:20px;">
  <li><code>app.py</code> – Archivo principal de la aplicación.</li>
  <li><code>requirements.txt</code> – Lista de dependencias necesarias.</li>
  <li><code>TelcoCustomerChurn.csv</code> – Dataset a cargar para su posterior análisis.</li>
  <li><code>README.md</code> – Documentación del proyecto.</li>
  <li><code>imagen_home.jpg</code> – Imagen de página Home.</li>
</ul>

<h2 style="color:#111827; font-family:Segoe UI, system-ui;">🚀 Instrucciones de ejecución</h2>
<ol style="color:#374151; font-size:14px; margin-left:20px;">
  <li>
    <strong>Instalar dependencias</strong><br />
    Verificar el archivo <code>requirements.txt</code> e instalar las librerías:
    <ul style="margin-top:4px; margin-left:20px;">
      <li>Pandas</li>
      <li>Streamlit</li>
      <li>Numpy</li>
      <li>Matplotlib</li>
      <li>Seaborn</li>
    </ul>
    <pre style="background:#0b1120; color:#e5e7eb; padding:10px; border-radius:6px; font-size:13px;">
pip install -r requirements.txt</pre>
  </li>
  <li style="margin-top:8px;">
    <strong>Ejecutar la aplicación</strong><br />
    En la consola (CMD), ejecutar:
    <pre style="background:#0b1120; color:#e5e7eb; padding:10px; border-radius:6px; font-size:13px;">
streamlit run app.py</pre>
  </li>
  </li>
  <li style="margin-top:8px;">
    <strong>NOTA: El archivo imagen_home.jpg debe encontrarse en la misma carpeta que la aplicación (app.py)</strong><br />
  </li>
</ol>
